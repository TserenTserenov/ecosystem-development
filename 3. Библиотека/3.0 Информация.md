
- **Библиотека** делится на две зоны:  
    – _curated_ (ручная, редактируемая людьми) — тексты, руководства, определения;  
    – _derived_ (автоматическая) — статистика, отчёты, вычисленные данные.
    
- **Единый глоссарий** и **онтология** понятий.
    
- **Версионирование** — через Git/Obsidian.



Используй плагин **Dataview** или **DataviewJS**, чтобы автоматически генерировать список заметок по тегу `#понятие`, `#теория`, `#источник`.
Например:
`table file.link as "Понятие", tags from "3_Библиотека/Понятия" sort file.name asc`
